# Disease Status Initialization Module for icm

This function sets the initial disease status on the network given the
specified initial conditions.

## Usage

``` r
init_status.icm(dat)
```

## Arguments

- dat:

  Main `icm_dat` class data object passed through `icm` simulations.

## Value

The updated `icm_dat` class main data object.

## See also

This is an initialization module for
