# Convert Timed Edgelist to Matrix of Toggles

Convert Timed Edgelist to Matrix of Toggles

## Usage

``` r
tedgelist_to_toggles(tedgelist)
```

## Arguments

- tedgelist:

  A timed edgelist, as produced by
  [`networkDynamic::as.data.frame.networkDynamic`](https://rdrr.io/pkg/networkDynamic/man/as.data.frame.networkDynamic.html).

## Value

The matrix of toggles corresponding to `tedgelist`.
