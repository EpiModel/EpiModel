# Delete Elements from Attribute List

Deletes elements from the main attribute list.

## Usage

``` r
delete_attr(dat, ids)
```

## Arguments

- dat:

  Either the `netsim_dat` class object passed through `netsim`
  simulations, or the main data object passed through `icm` simulations.

- ids:

  ID numbers to delete from the list.

## Value

The updated `netsim_dat` main list object.
